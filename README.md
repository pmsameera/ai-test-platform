# AI Test Management Platform

A production-grade test management and execution platform built using FastAPI, React, PostgreSQL, Docker, Playwright, and Google Cloud Platform.

## Status

🚧 In Development

## Tech Stack

- FastAPI
- React
- PostgreSQL
- Docker
- Playwright
- Pytest
- Google Cloud

## Structure

- `backend/` - Backend service code
- `frontend/` - Frontend application code
- `infrastructure/` - Infrastructure and deployment definitions
- `docs/` - Project documentation
- `.github/` - GitHub Actions workflows and repository metadata
- `docker-compose.yml` - Local development container orchestration
